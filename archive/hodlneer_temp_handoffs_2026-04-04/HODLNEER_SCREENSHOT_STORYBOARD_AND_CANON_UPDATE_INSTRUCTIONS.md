## Core Screenshot Principle

> The screenshot sequence should sell Hodlneer as a premium identity artifact product in under ten seconds.

That means the sequence must communicate, fast:
1. proof
2. badge desirability
3. reveal energy
4. wallpaper payoff
5. privacy intelligence
6. optional HDL connection

If the screenshots look like “an app with some crypto screens,” the sequence failed.

## Screenshot Order Logic

The current sequence is strong because it follows this exact user psychology:

1. **What is this?**
2. **Why is it different?**
3. **Why is it legitimate?**
4. **Why would I want it on my phone?**
5. **Will this force me to expose too much?**
6. **What else can it become?**

That is the correct mental path.

Do not shuffle the order casually.

---

## Visual Production Guidance

### Every screenshot should preserve:
- badge-first hierarchy
- premium dark-luxury or high-contrast feel
- clean copy
- readable typography
- enough UI to show product reality
- enough composition to feel like a campaign asset

### Typography Guidance
Headline copy should be:
- bold
- short
- high-contrast
- emotionally direct

Subheadline copy should be:
- shorter than landing page language
- easier than manifesto language
- sharper than generic marketing fluff

### Framing Guidance
Do not crop so tightly that the screen loses context.
Do not zoom out so far that the badge loses impact.

The product should feel real and elevated at the same time.

---

## Motion-to-Still Guidance

Because Hodlneer includes reveal and wallpaper motion, screenshot stills must be chosen carefully.

### Good still choices
- post-impact reveal freeze
- badge in stable premium idle state
- wallpaper at its strongest composed moment
- privacy screen with meaningful but uncluttered controls

### Bad still choices
- blurry transition moments
- empty loader frames
- tiny unreadable UI states
- a random frame where motion made sense but the still does not

Stills must feel intentional.
No lazy frame grabs.

## What Good Looks Like

A strong screenshot set should make someone think:
- “I get this immediately.”
- “That looks premium.”
- “This is not a generic crypto app.”
- “I want to see my badge.”
- “The wallpaper angle is real, not tacked on.”
- “I understand HDL is connected, but I also understand Hodlneer by itself.”

## What Must Never Happen

The screenshot set must never feel like:
- random app captures with slogans
- a designer trying too hard to be cinematic
- HDL promo art disguised as Hodlneer screenshots
- crypto technical documentation
- a wallpaper app with no proof logic
- a proof app with no emotional payoff

That would kill conversion.
